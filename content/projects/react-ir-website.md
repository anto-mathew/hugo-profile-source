## Overview
A modern **React.js Investor Relations (IR) website** for corporate financial reporting, regulatory disclosures, and investor communications.

## Features
- Dynamic pages generated using JSON routing
- Reusable components for results, financial statements, filings
- API integrated for disclosures such as:
  - Annual Reports
  - Earnings Releases
  - Press Announcements
- SEO-ready & mobile-optimized layout
- Light and fast UI using modern React hooks

## Responsibilities
- Architected the full UI component system
- Implemented routing using JSON-driven configuration
- Integrated with backend IR API
- Created reusable templates for financial widgets

## Tech Stack
**React.js, Vite, TailwindCSS, REST API**.
