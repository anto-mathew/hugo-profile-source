## Overview
A lightweight analytics dashboard built using **Svelte**, focusing on fast rendering and dynamic components.

## Features
- Dynamic components with `#each` and conditional rendering
- Locale-based UI switching (`tc`, `sc`, `en`)
- Custom tooltip implementation without Bootstrap
- Component-level TypeScript validation
- API integration for charts and metrics

## Responsibilities
- Architecture of component structure
- Implemented reusable components with props
- Added locale switching logic
- Created tooltip system
- Worked with dynamic lists and API data

## Tech Stack
**Svelte**, **TypeScript**, **Fetch API**.
