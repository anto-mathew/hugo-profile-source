## Overview
A full WordPress website built using the Bedrock structure and the Sage Blade-based theme to achieve a modern, modular, and secure development workflow.

## Features
- Blade templating system  
- Composer-managed WordPress  
- REST API with custom fields  
- Custom menu click tracking  
- Advanced ACF + REST integration  
- Clean MVC-like folder structure  

## Responsibilities
- Extended REST API responses
- Built custom post types and fields
- Implemented frontend behaviour with JS tracking
- Integrated Sage templates with Blade
- Setup Bedrock environment for both local and production

## Tech Stack
WordPress Bedrock, Sage, PHP, Composer, Blade, JavaScript.
